# İTÜ Takvimci

İTÜ OBS verileriyle ders programı oluşturmaya yardımcı olan basit bir web aracı.

## Özellikler
- Bölüm filtresiyle dersleri daraltma
- Birden fazla takvim oluşturma
- Takvim görüntüsünü indirme
- Alternatif CRN programları hesaplama
- Seçimleri tarayıcıda saklama

## Kullanım
1. Ana Dal / Yan Dal filtresi seçin.
2. Ders kodu, ders ve CRN seçin.
3. Takvim otomatik oluşur.

## Alternatif CRN
"Alternatif CRN Hesapla" ile çakışmasız alternatif programları gezebilirsiniz.
